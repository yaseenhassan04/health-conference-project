# ✅ قائمة التحقق الاحترافية للنشر

## 📋 قبل البدء بالتطوير

- [ ] تثبيت Node.js v18+
- [ ] تثبيت Git
- [ ] إنشاء حساب في GitHub (للتحكم الإصدار)
- [ ] إنشاء حساب في Vercel أو منصة نشر أخرى

## 🗄️ إعداد قاعدة البيانات

- [ ] تشغيل `npm install`
- [ ] تشغيل `npm run db:init` (أو `node scripts/setup.bat`)
- [ ] التحقق من إنشاء `dev.db`
- [ ] اختبار قاعدة البيانات: `npm run db:studio`

## 🔐 الأمان (قبل الإنتاج)

### ⚠️ حساب المسؤول
- [ ] تغيير كلمة المرور الافتراضية (غيّر `admin` في `scripts/init-db.js`)
- [ ] إضافة تجزئة (hashing) لكلمات المرور (استخدم `bcrypt`)
- [ ] إضافة مصادقة متقدمة (JWT أو OAuth)

### 📧 البريد الإلكتروني
- [ ] إنشاء حساب Gmail منفصل للإرسال
- [ ] إعداد كلمة مرور التطبيق (App Password)
- [ ] تحديث `.env` مع بيانات البريد الصحيحة
- [ ] اختبار إرسال بريد تجريبي

### 🌐 الإنترنت والشهادات
- [ ] شراء نطاق (domain)
- [ ] إعداد شهادة SSL (HTTPS)
- [ ] إعداد CDN (اختياري)

## 🔧 الأوامر المهمة

```bash
# التطوير
npm run dev

# البناء للإنتاج
npm run build
npm start

# إدارة قاعدة البيانات
npm run db:init      # إنشاء قاعدة جديدة
npm run db:push      # إضافة النموذج
npm run db:studio    # فتح واجهة إدارة

# إعادة تشغيل قاعدة البيانات (حذر: يحذف البيانات)
npm run db:reset
```
